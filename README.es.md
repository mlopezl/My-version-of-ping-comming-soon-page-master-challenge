El desafío me ayudó a mejorar mis habilidades en **HTML, CSS y JavaScript** al construir una landing page responsiva con un sistema funcional de validación de correo electrónico.

## Resumen
Este proyecto es una **página de suscripción "próximamente"** para un servicio ficticio llamado Ping.  
Los usuarios pueden ingresar su correo electrónico para suscribirse y recibir una notificación cuando el servicio sea lanzado.

La interfaz incluye:
- Un diseño limpio y responsivo  
- Validación de correo electrónico en tiempo real al enviar el formulario  
- Retroalimentación visual de error  
- Íconos de redes sociales con estados hover  

## El Desafío
Los usuarios deberían poder:

- Ingresar un correo electrónico en el campo de suscripción  
- Ver mensajes de error cuando el correo tiene un formato inválido  
- Ver estados hover en elementos interactivos  
- Visualizar el diseño correctamente en todos los tamaños de pantalla

## Mi Proceso
1. Comencé creando la estructura básica con **HTML semántico**.  
2. Construí un diseño responsivo basado en componentes usando **Flexbox**.  
3. Añadí un estilo visual consistente mediante **propiedades personalizadas (variables) en CSS**.  
4. Estilicé el formulario, los estados de los inputs y los íconos sociales con estilos hover y active.  
5. Implementé la validación del correo electrónico usando **expresiones regulares en JavaScript**.  
6. Apliqué retroalimentación visual de error y éxito mediante manipulación dinámica de clases.  
7. Finalicé el comportamiento responsivo para pantallas pequeñas con media queries.

## Lo que aprendí
- Cómo crear un **flujo de validación de formularios** usando expresiones regulares y actualización condicional de clases.  
- Cómo crear una interfaz reutilizable y escalable usando **variables de CSS**.  
- Cómo gestionar **estados de error y éxito** visualmente con CSS y manipulación del DOM.  
- Cómo estructurar un diseño responsivo utilizando **flexbox y media queries**.  
- Cómo implementar efectos hover para mejorar la retroalimentación en la interacción del usuario.
